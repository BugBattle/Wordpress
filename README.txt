== Description ==

Bugbattle helps developers build the best software faster. It is your affordable in-app bug reporting tool for apps, websites and industrial applications.

No more wasting time trying to reproduce a bug. Bugbattle reports automatically contain a replay video, session data, logs and more. Even better: You can add custom data to your bug details.

== Installation ==

Let's get started with the Bugbattle plugin for WordPress

e.g.

1. Upload `bugbattle.zip` to the `/wp-content/plugins/` directory and extract it
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
